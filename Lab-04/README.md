
# EDA Lab 4

This repository contains the implementation of **Power Transformations (Box-Cox and Yeo-Johnson)** on the Concrete Strength dataset.

## Contents

* `EDA.LAB4.ipynb` – Jupyter Notebook
* `concrete_data.csv` – Dataset
* `README.md` – Documentation

## Objectives

* Apply Box-Cox and Yeo-Johnson transformations
* Analyze feature distributions
* Train and evaluate Linear Regression models
* Compare model performance before and after transformation

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Results

The project demonstrates how power transformations can reduce skewness and improve machine learning model performance.
