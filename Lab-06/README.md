# EDA Lab 6

This repository contains **Exploratory Data Analysis (EDA) Lab 6** implemented using Python. The analysis is performed on the **country_wise_latest.csv** dataset to explore COVID-19 statistics across different countries.

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed

* Dataset loading and inspection
* Statistical summary of data
* Top countries by confirmed cases
* Top countries by deaths
* Bar charts and line plots
* Scatter plot of confirmed cases vs deaths
* Death rate analysis
* Pie chart visualization

## Dataset

* `country_wise_latest.csv`

## Files

* `EDA.LAB6.ipynb` – Jupyter Notebook containing the complete analysis.
* `country_wise_latest.csv` – Dataset used for the analysis.

## Objective

To understand and visualize global COVID-19 trends using exploratory data analysis techniques.
